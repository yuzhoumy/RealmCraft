這是我先前製作的伺服器 領域世界 | RealmCraft 的插件。
這個伺服器現在已經沒有營運，因此這個插件也沒有再維護了。

領域世界 | RealmCraft 的插件：

[RealmCraft](https://github.com/yuzhoumy/RealmCraft/) ：主要插件，負責大部分伺服器的特殊機制。
[RealmTrader](https://github.com/yuzhoumy/RealmTrader/) ：GUI商店插件，大部分套用在NPC上。
[ResourcePoint](https://github.com/yuzhoumy/ResourcePoint/) ：資源點插件，負責資源點刷新、生成的機制。
[RCChest](https://github.com/yuzhoumy/RCChest/) ：資源箱插件，負責資源點所生成的資源箱分配獎品的系統。
[PortalSystem](https://github.com/yuzhoumy/PortalSystem/) ：傳送門插件，管理傳送門的所有功能。
